# Текстовое решение задачи

1. По условию у нас есть массив → *ArrayString[size]*
2. Задаём переменные i, j, NewArray:    
    * **i** - индекс массива ArrayString.
    * **j** - индекс массива NewArray.
    * **NewArray** - будущей новый массив.
3. Задаём условие,когда остановится повторение цикла: *i>size*
    * **i** - индекс массива ArrayString.
    * **size** - размер массива (его длинна, количество элементов).
4. Задаём условие, индекс нашего массива <= 3 ?
    * Yes → Тогда в массив NewArray с индексом j, мы добавляем значение из массива ArrayString 

        → увеличиваем индекс j на 1.

        → увеличиваем индекс i на 1.
    * No → увеличиваем индекс i на 1.
5. После того, как i станет >= size, мы покажем полученный массив.
